A Propos de moi

Salut, je suis Abderrahim Khantouch.

 

Actuellement, je suis un étudiant suivant une formation en bachelor en alternance à l'ESN de Castres, option Concepteur Développeur Web Full Stack, l'option spécialisée dans le développement web, l'informatique de gestion et la gestion de projets informatique.

 

L'informatique n'a pas été un domaine qui m'a tout de suite intéressé, avant cette formation j'ai effectué après mon année de terminale une année de BTS CRSA (Conception et Réalisation de Systèmes Automatisés), juste après avoir reçu mon bac pro en Bac Pro MEI (Maintenance des Équipements Industriels).

 

Cette année de BTS, je ne l'avais pas terminé par manque d'intérêt pour ce que je faisais dans cette formation.

 

J'ai découvert cette année là que l'informatique m'intéressait bien plus et donc avec Parcoursup, j'ai changé de parcours pour intégrer le BTS SIO, le BTS qui lorsque j'ai voulu changer de formation paraissait le plus approprié pour en apprendre plus sur l'informatique.

 

Et depuis, je poursuis mes études en informatique en vu d'avoir les diplômes et compétences nécessaires pour commencer une carrière de développeur informatique.

Les outils et langages que j'ai utilisé pendant mon parcours scolaire et professionnel.

Langage de programmation utilisé pour créer des sites Web dynamiques et des applications Web
Le CSS sert à définir un ensemble de règles stylistiques à l'intérieur de pages web codées en HTML ou en XML
Le HyperText Markup Language, généralement abrégé HTML est un langage de balisage conçu pour écrire des pages web
Le JavaScript est un langage de script léger, orienté frontend, principalement connu comme un langage de script pour des pages web
Java est un langage de programmation orienté backend, derrière de nombreux logiciels d'entreprise

VirtualBox est un logiciel de virtualisation open source qui permet aux utilisateurs de créer et d'exécuter des machines virtuelles
